# fcitx5-vinput site

Starlight-based static site for the project landing page and install page.

## Commands

Run commands from `site/`:

| Command | Action |
| :------ | :----- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server |
| `npm run build` | Build production site |
| `npm run preview` | Preview the built site |
| `npm run check` | Run Astro type/content checks |

## Current scope

- Starlight site scaffold
- English + Simplified Chinese locale routing
- GitHub Pages workflow
- Landing page
- Install page
